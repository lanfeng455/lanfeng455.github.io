/ 关于我

你好，我是洪鑫裕，FZU数字媒体技术大四学生，主修游戏设计。
我始终相信，游戏是当代最迷人的数字娱乐媒介。它是代码逻辑、文学感性与视觉张力交融的十字路口。


在校期间，我主要使用Unity C#进行开发，参与过多次游戏制作大赛及Game Jam，独立全栈开发过若干项目。

同时，我习惯用镜头捕捉城市的人文与风景，用文字把感性的思考转化为客观的叙事。这些跨界的练习让我对用户体验、情感表达有更细腻的理解。

目前我正寻找一份Unity客户端实习，希望能将我的技术热情与艺术感知结合起来，将跨学科和人本主义的思想带入到有温度的创作中。

我的技术栈：C# | Unity | RPGMaker | Java | Python | Adobe系
